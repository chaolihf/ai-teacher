# 计算长方形面积的 C 语言程序

## 任务描述
使用 C 语言创建一个计算长方形面积的程序，包含完整的编译和测试流程。

## 一、源程序代码

```c
#include <stdio.h>
#include <locale.h>

int main() {
    // 设置本地化为中文，以正确显示中文字符
    setlocale(LC_ALL, "");

    double length, width, area;

    printf("请输入长方形的长：");
    if (scanf("%lf", &length) != 1) {
        printf("输入失败！\n");
        return 1;
    }

    printf("请输入长方形的宽：");
    if (scanf("%lf", &width) != 1) {
        printf("输入失败！\n");
        return 1;
    }

    area = length * width;

    printf("长方形的面积为：%.2f\n", area);

    return 0;
}
```

### 代码说明
- 使用 `double` 类型存储长、宽和面积，支持小数运算
- `setlocale(LC_ALL, "")` 用于支持中文本地化
- 使用 `if (scanf(...) != 1)` 检查输入是否成功
- `%.2f` 格式化输出，保留两位小数

## 二、编译命令

### UTF-8 编码（推荐，需切换控制台编码）
```bash
gcc rectangle_area.c -o rectangle_area -fexec-charset=UTF-8
```

### GBK 编码（Windows 默认，无需切换编码）
```bash
gcc rectangle_area.c -o rectangle_area -fexec-charset=GBK
```

## 五、常见问题

### 问题：中文显示乱码
**原因**：Windows 命令行默认使用 GBK 编码，而源文件使用 UTF-8 编码。

**解决方法**：
1. **临时方案**：运行程序前执行 `chcp 65001` 切换为 UTF-8 编码
2. **使用 GBK 编译**：使用 `-fexec-charset=GBK` 参数重新编译
