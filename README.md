# 🌈 颜色教学实验室

一个基于 React 的互动式 RGB 色彩原理教学工具，帮助学生直观理解颜色的构成和变化规律。

## 📋 项目需求总结

### 背景
本项目旨在通过互动实验的方式，帮助学生理解 RGB 色彩模型的基本原理，包括：
- 颜色的三原色构成
- 像素颜色的数字化表示
- 图像处理中的颜色变换

### 核心功能需求

#### 1. RGB 调色板 🎨
- **需求描述**：通过设置不同的 R、G、B 值来显示不同的颜色
- **功能实现**：
  - 三个滑块分别控制红、绿、蓝通道（0-255）
  - 实时显示混合后的颜色
  - 同时显示 RGB 和 HEX 格式的颜色值
  - 提供常见颜色组合的参考表

#### 2. 图片取色器 🖱️
- **需求描述**：通过鼠标选择一张图片上的点来显示对应的 R、G、B 值
- **功能实现**：
  - 支持上传图片（JPG、PNG、GIF 等格式）
  - 提供 5 张教学演示图片快速选择
  - 鼠标移动时实时显示像素 RGB 值
  - 点击可固定当前像素颜色信息
  - 显示位置坐标、RGB 值、HEX 值和透明度

#### 3. 图像处理实验 🔬
- **需求描述**：设计图像反转、黑白化的实验，查看对应的 RGB 值变化
- **功能实现**：
  - **图像反转（反色）**：将每个像素的 RGB 值取反（新值 = 255 - 原值）
  - **黑白化（灰度）**：将彩色图转换为灰度图（灰度值 = 0.299×R + 0.587×G + 0.114×B）
  - 原图与处理后图片并排对比
  - 点击任意像素点对比处理前后的 RGB 值变化
  - 显示变化差值

## 🛠️ 技术栈

- **框架**：React 19

- **语言**：JavaScript (ES6+)
- **样式**：CSS3（渐变、动画、响应式布局）
- **图像处理**：HTML5 Canvas API

## 📖 使用说明

### 1. RGB 调色板
1. 拖动 R、G、B 三个滑块调整颜色值
2. 观察颜色显示区域的变化
3. 记录不同颜色组合的效果

### 2. 图片取色器
1. 点击示例图片按钮或上传自己的图片
2. 将鼠标移动到图片上查看像素 RGB 值
3. 点击鼠标固定当前像素信息

### 3. 图像处理实验
1. 选择示例图片或上传图片
2. 选择"图像反转"或"黑白化"效果
3. 点击原图或处理后的图片上的点
4. 对比查看 RGB 值的变化

## 📚 教学知识点

### RGB 色彩模型
- **三原色**：红 (Red)、绿 (Green)、蓝 (Blue)
- **取值范围**：每种颜色 0-255，共 256 级
- **颜色总数**：256 × 256 × 256 = 16,777,216 种颜色

### 常见颜色组合
| 颜色 | RGB 值 |
|------|--------|
| 黑色 | (0, 0, 0) |
| 白色 | (255, 255, 255) |
| 红色 | (255, 0, 0) |
| 绿色 | (0, 255, 0) |
| 蓝色 | (0, 0, 255) |
| 黄色 | (255, 255, 0) |
| 青色 | (0, 255, 255) |
| 品红 | (255, 0, 255) |
| 灰色 | (128, 128, 128) |

### 图像处理原理
- **图像反转**：`新值 = 255 - 原值`
- **黑白化**：`灰度值 = 0.299×R + 0.587×G + 0.114×B`
  - 系数考虑了人眼对不同颜色的敏感度
  - 绿色权重最高，蓝色最低

## 📝 开发说明

### 组件设计
- **RgbPalette**：状态管理 + 滑块控制
- **ImageColorPicker**：Canvas 绘图 + 事件处理
- **ImageProcessing**：双 Canvas 对比 + 像素级图像处理

### 关键技术点
- 使用 `useEffect` 处理图片加载和 Canvas 绘制
- 通过 `getImageData` 获取像素数据
- 响应式布局适配不同屏幕尺寸
